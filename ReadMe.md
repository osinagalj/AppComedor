Restaurant App
===============

A FoodApp built using Android Studio.

 - #### User Application

# Topics
- Linear Layout
- Card View
- Adapters
- Objects Serializable
- Comunication Fragment-Fragment & Fragment-Activity
- RecyclerView
- Android Icons
- Drawe Menu
- RecyclerView Details
- NestedRecyclerView
- Fragments
- ViewPager
- RecyclerView click on Card
- NotifyItemRemoved
- AdobeXD for UI/UX
